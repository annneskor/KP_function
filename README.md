# Курсовой проект

## Цели и задачи

**Целью** курсового проекта является разработка консольной программы на языке С, которая позволяет пользователю выполнять операции над кусочно-заданной функцией.

**Задачи** курсового проекта:
* проанализировать математическое описание функции и определить её область допустимых значений;
* реализовать основную функцию f(x) с обработкой различных интер-валов аргумента;
* разработать пользовательские функции для выполнения каждой из заявленных операций;
* обеспечить валидацию входных данных и корректную обработку ошибок в каждой функции;
* создать удобное консольное меню в главной функции, позволяющее выбирать необходимую операцию;
* реализовать запись результатов табулирования в текстовый файл с возможностью перезаписи или добавления;
* провести тестирование программы на различных наборах данных для проверки корректности работы всех режимов.

### Код программы
```c
#include <stdio.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "");
    int arr[7];
    int sum = 0, count = 0;

    printf("Введите 7 целых чисел (положительных и отрицательных):\n");

    for (int i = 0; i < 7; i++) {
        printf("Элемент %d: ", i + 1);
        scanf("%d", &arr[i]);
    }

    // Вычисление среднего значения элементов с четными индексами (кроме нулевого)
    for (int i = 2; i < 7; i += 2) {
        sum += arr[i];
        count++;
    }

    printf("\nВведенный массив: ");
    for (int i = 0; i < 7; i++) {
        printf("%d ", arr[i]);
    }

    printf("\nЭлементы с четными индексами (кроме 0-го): ");
    for (int i = 2; i < 7; i += 2) {
        printf("%d ", arr[i]);
    }

    if (count > 0) {
        double average = (double)sum / count;
        printf("\nСреднее значение: %.2f\n", average);
    }
    else {
        printf("\nНет элементов для вычисления среднего значения.\n");
    }

    return 0;
}
```

#include <stdio.h>
#include <locale.h>


int main() {
    setlocale(LC_ALL, "RUS");

    int A, B, C;
    int hypothesis;

    printf("=== ПРОВЕРКА ГИПОТЕЗЫ ===\n");
    printf("Введите три целых числа (A, B, C): ");

    scanf("%d %d %d", &A, &B, &C);

    // Проверяем, что все три числа кратны трём

    printf("Гипотеза подтверждена (1 - да, 0 - нет): %d\n", hypothesis);

    return 0;
}
