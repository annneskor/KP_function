# Курсовой проект

## Цели и задачи

**Целью** курсового проекта является разработка консольной программы на языке С, которая позволяет пользователю выполнять операции над кусочно-заданной функцией.

**Задачи** курсового проекта:
* проанализировать математическое описание функции и определить её область допустимых значений;
* реализовать основную функцию f(x) с обработкой различных интер-валов аргумента;
* разработать пользовательские функции для выполнения каждой из заявленных операций;
* обеспечить валидацию входных данных и корректную обработку ошибок в каждой функции;
* создать удобное консольное меню в главной функции, позволяющее выбирать необходимую операцию;
* реализовать запись результатов табулирования в текстовый файл с возможностью перезаписи или добавления;
* провести тестирование программы на различных наборах данных для проверки корректности работы всех режимов.
